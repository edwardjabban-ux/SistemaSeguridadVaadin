
# Sistema de Seguridad Integral

## Tecnologías
- Java 17
- Vaadin
- Spring Boot
- PostgreSQL

## Ejecutar Proyecto
1. Crear BD PostgreSQL
2. Ejecutar database.sql
3. Configurar application.properties
4. Abrir en IntelliJ
5. Ejecutar SistemaSeguridadApplication

## Funcionalidades
- Registrar usuarios
- Crear solicitudes de servicios
- Agregar multiples servicios al catalogo
- Calcular costo total de una solicitud
- Realizar pago
- Asignar escoltas y vehiculos
- Activar emergencia
- Ver historial de solicitudes
- Calificar servicio

## Requisitos no funcionales visibles en la app
- Seguridad de datos
- Alta disponibilidad
- Escalabilidad
- Interfaz amigable
- Tiempo de respuesta rapido
